Functionalities:

Image Resizing:

  • This feature allows users to resize images to different dimensions. Users can upload multiple files and choose from predefined size options. Images that do not have the specific suffixes (_s.png, _n.png) will be resized with nearest interpolation and the rest resized with bilinear. The application also supports applying diffusion dithering for images that don't have specific suffixes (_s.png, _n.png).

    Key Features:

  • Multiple file upload with a limit of 5000 images at a max of 1GB in total.
  • Preset size options for resizing.
  • Different resize methods for each of the three texture groups.
  • Optional diffusion dithering and compression with minimal detail loss.
  • Temporary storage of processed images and auto-deletion after a 1-minute duration.
  • After processing, users can download their images. The images are compressed into a zip file for convenience.
